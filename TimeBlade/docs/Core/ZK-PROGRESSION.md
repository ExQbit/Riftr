# Zeitklingen: Progression & Pacing

## XP-Boost-System

### Tägliches XP-Limit
- **60.000 XP pro Tag** aus allen Quellen (Kämpfe, Quests, Events)
- **Reset**: Täglich um 00:00 UTC
- **Übertrag**: Nicht genutztes XP verfällt

### XP-Boost-Stufen
| XP-Bereich | Multiplikator | Beschreibung |
|------------|--------------|-------------|
| 0-10.000 | ×4.0 | Schnellstart-Boost |
| 10.001-22.500 | ×3.0 | Aktiver Spieler-Boost |
| 22.501-37.500 | ×2.0 | Standard-Boost |
| 37.501-60.000 | ×1.0 | Basis-XP-Rate |

### XP-Boost-Quellen
- **Gameplay-Belohnungen**: Automatisch durch Aktivitäten freigeschaltet
- **Tagesquests**: 3 abgeschlossene Quests = +50% Boost für 2h
- **Events**: Temporäre Multiplikatoren während Events
- **Premium**: Battle Pass bietet +25% XP-Bonus (nicht P2W)

## Klassenstufen-System

### Grundmechanik
- **25 Klassenstufen** mit exponentiell steigenden XP-Anforderungen
- **Kartenlevel-Limit**: Kartenlevel ≤ Klassenstufe × 2
- **Klassenspezifische Boni** bei Stufe 5, 10, 15, 20, 25
- **Integration mit XP-Boost**: Alle XP-Quellen profitieren von Boosts

```python
        "world_completion": {"base": 8000, "scaling": 1.0}
    }
}

def calculate_xp_gain(source_type, source_name, player_level, world_level, player_id=None):
    """Berechnet XP-Gewinn für eine Aktivität mit XP-Boost-System"""
    source_data = XP_SOURCES[source_type][source_name]
    base_xp = source_data["base"]
    scaling = source_data["scaling"]
    
    # Level-Scaling
    level_multiplier = 1.0 + (player_level - 1) * 0.05  # +5% pro Level
    
    # World-Scaling
    world_multiplier = 1.0 + (world_level - 1) * 0.3  # +30% pro Welt
    
    # Anti-Grinding Protection (niedrigere XP für Over-Leveled Content)
    recommended_level = world_level * 5  # Welt 1 = Level 5, etc.
    if player_level > recommended_level + 3:
        level_multiplier *= 0.7  # 30% Penalty für Over-Leveling
    
    # Basis-XP berechnen
    base_final_xp = int(base_xp * scaling * level_multiplier * world_multiplier)
    base_final_xp = max(10, base_final_xp)  # Minimum 10 XP
    
    # XP-Boost-System anwenden
    if player_id:
        boosted_xp = apply_xp_boost_system(player_id, base_final_xp)
        return boosted_xp
    
    return base_final_xp

def apply_xp_boost_system(player_id, base_xp):
    """Wendet das tägliche XP-Boost-System an"""
    player_data = get_player_daily_xp_data(player_id)
    current_daily_xp = player_data["current_daily_xp"]
    
    # Prüfe tägliches Limit
    if current_daily_xp >= 60000:
        return 0  # XP-Limit erreicht
    
    # Bestimme aktuellen Boost-Multiplikator
    boost_multiplier = get_xp_boost_multiplier(current_daily_xp)
    
    # Berechne geboostete XP
    boosted_xp = int(base_xp * boost_multiplier)
    
    # Prüfe ob XP das tägliche Limit überschreiten würde
    if current_daily_xp + boosted_xp > 60000:
        boosted_xp = 60000 - current_daily_xp
    
    # Update tägliche XP
    update_daily_xp(player_id, boosted_xp)
    
    return boosted_xp

def get_xp_boost_multiplier(current_daily_xp):
    """Bestimmt XP-Boost-Multiplikator basierend auf täglichem XP"""
    if current_daily_xp < 10000:
        return 4.0  # Schnellstart-Boost
    elif current_daily_xp < 22500:
        return 3.0  # Aktiver Spieler-Boost
    elif current_daily_xp < 37500:
        return 2.0  # Standard-Boost
    else:
        return 1.0  # Basis-XP-Rate
```

```python
WORLD_REQUIREMENTS = {
    1: {"min_class_level": 1, "cards_at_level": 0, "story_completion": 0},
    2: {"min_class_level": 5, "cards_at_level": {"level": 10, "count": 3}, "story_completion": 1},
    3: {"min_class_level": 11, "cards_at_level": {"level": 15, "count": 5}, "story_completion": 2},
    4: {"min_class_level": 16, "cards_at_level": {"level": 25, "count": 3}, "story_completion": 3},
    5: {"min_class_level": 21, "cards_at_level": {"level": 35, "count": 2}, "story_completion": 4}
}

def check_world_unlock(player_data, target_world):
    """Prüft ob Spieler eine Welt freischalten kann"""
    requirements = WORLD_REQUIREMENTS[target_world]
    
    # Klassenlevel prüfen
    if player_data["class_level"] < requirements["min_class_level"]:
        return False, f"Klassenlevel {requirements['min_class_level']} erforderlich"
    
    # Karten-Level prüfen
    if "cards_at_level" in requirements and requirements["cards_at_level"]:
        req_level = requirements["cards_at_level"]["level"]
        req_count = requirements["cards_at_level"]["count"]
        actual_count = count_cards_at_level(player_data["cards"], req_level)
        
        if actual_count < req_count:
            return False, f"{req_count} Karten auf Level {req_level}+ erforderlich"
    
    # Story-Completion prüfen
    if player_data["completed_worlds"] < requirements["story_completion"]:
        return False, f"Welt {requirements['story_completion']} muss abgeschlossen sein"
    
    return True, "Freischaltung möglich"
```

```python
def calculate_hook_timing(player_id):
    """Berechnet optimales Timing für Progression-Hooks"""
    
    player_data = get_player_data(player_id)
    session_count = player_data["session_count"]
    avg_session_length = player_data["avg_session_minutes"]
    last_hook = player_data["last_progression_hook"]
    
    # Adaptive Hook-Timing basierend auf Spielverhalten
    if avg_session_length < 15:  # Kurze Sessions
        hook_interval = 2  # Häufigere Hooks
    elif avg_session_length > 60:  # Lange Sessions
        hook_interval = 4  # Seltener, aber bedeutungsvoller
    else:
        hook_interval = 3  # Standard
    
    # Zeit seit letztem Hook
    time_since_last = (datetime.now() - last_hook).total_seconds() / 3600
    
    if time_since_last >= hook_interval:
        return True, get_next_appropriate_hook(player_data)
    
    return False, None

def trigger_progression_hook(player_id, hook_type):
```

### Retention-Optimierung
```python
def calculate_retention_risk(player_data):
    """Berechnet Retention-Risiko für proaktive Intervention"""
    
    risk_factors = {}
    
    # Session-Frequenz
    days_since_last_session = (datetime.now() - player_data["last_session"]).days
    if days_since_last_session > 3:
        risk_factors["absence"] = min(1.0, days_since_last_session / 7)
    
    # Progression-Stagnation
    hours_without_levelup = player_data["hours_since_last_levelup"]
    expected_levelup_interval = 2  # Erwartung: Level-up alle 2h
    if hours_without_levelup > expected_levelup_interval * 2:
        risk_factors["stagnation"] = min(1.0, hours_without_levelup / 10)
    
    # Material-Frustration
    failed_evolutions = player_data.get("failed_evolution_attempts", 0)
    if failed_evolutions > 2:
        risk_factors["frustration"] = min(1.0, failed_evolutions / 5)
    
    # Hook-Timing
    time_since_last_hook = player_data["hours_since_last_hook"]
    if time_since_last_hook > 5:  # Zu lange ohne Progression-Hook
        risk_factors["engagement"] = min(1.0, time_since_last_hook / 12)
    
    total_risk = sum(risk_factors.values()) / len(risk_factors) if risk_factors else 0
    
    return {
        "total_risk": total_risk,
        "factors": risk_factors,
        "intervention_recommended": total_risk > 0.6
    }

def suggest_retention_intervention(risk_analysis):
    """Schlägt Retention-Maßnahmen vor"""
    
    if risk_analysis["total_risk"] < 0.3:
        return None
    
    interventions = []
    
    if "stagnation" in risk_analysis["factors"]:
        interventions.append({
            "type": "material_gift",
            "description": "3 Zeitkerne + 1 Elementarfragment",
            "trigger": "login_bonus"
        })
    
    if "frustration" in risk_analysis["factors"]:
        interventions.append({
            "type": "guaranteed_success",
            "description": "Nächste Evolution 100% Erfolgsrate",
            "trigger": "evolution_attempt"
        })
    
    if "engagement" in risk_analysis["factors"]:
        interventions.append({
            "type": "special_event",
            "description": "2x XP Event für 24h",
            "trigger": "immediate"
        })
    
    return interventions
```

```python
def generate_daily_quests(player_data):
    """Generiert personalisierte Tagesquests"""
    
    player_level = player_data["class_level"]
    current_world = player_data["current_world"]
    play_style = analyze_play_style(player_data)
    
    quest_pool = []
    
    # Basis-Quests (immer verfügbar)
    quest_pool.extend([
        create_combat_quest(current_world, player_level),
        create_class_mechanic_quest(player_data["class"], player_level),
        create_material_gathering_quest(current_world)
    ])
    
    # Spielstil-spezifische Quests
    if play_style["prefers_long_fights"]:
        quest_pool.append(create_endurance_quest())
    elif play_style["prefers_quick_fights"]:
        quest_pool.append(create_speed_quest())
    
    # Progression-spezifische Quests
    if has_evolution_ready_cards(player_data):
        quest_pool.append(create_evolution_quest())
    
    if needs_material_boost(player_data):
        quest_pool.append(create_boost_quest(get_bottleneck_material(player_data)))
    
    # Wähle 3 Quests aus Pool (keine Duplikate)
    selected_quests = random.sample(quest_pool, min(3, len(quest_pool)))
    
    return selected_quests

def create_combat_quest(world, player_level):
    """Erstellt kampfbezogene Quest"""
    enemy_count = 5 + (world - 1) * 2  # Skaliert mit Welt
    xp_reward = 800 + (player_level - 1) * 40
    
    return {
        "id": generate_quest_id(),
        "type": "combat",
        "description": f"Besiege {enemy_count} Gegner in Welt {world}",
        "requirements": {"enemy_kills": enemy_count, "world": world},
        "rewards": {
            "xp": xp_reward,
            "time_cores": random.randint(2, 4)
        },
        "expires_in_hours": 24
    }
```

## Dynamisches Events-System

### Event-Kategorien

#### Blitz-Events (15-30 Min)
- **Häufigkeit**: Alle 2-3 Stunden
- **Fokus**: Schnelle Belohnungen für aktive Spieler
- **Beispiele**: 
  - **Zeitrausch**: +200% XP für alle Kämpfe
  - **Materialregen**: +150% Droprate für alle Materialien
  - **Perfekte Zeit**: Bonus-Belohnungen für Kämpfe <20s verbleibend

#### Tages-Events (24-48h)
- **Häufigkeit**: 2-3 pro Woche
- **Fokus**: Thematische Herausforderungen mit Story-Elementen
- **Beispiele**:
  - **Feuer-Zeitalter**: DoT-Schäden +100%, Feuer-Evolutionen 50% günstiger
  - **Frostzeit**: Slow-Effekte +50%, Eis-Karten +25% Effektivität
  - **Gewittersturm**: Ketteneffekte +1 Ziel, Blitz-Karten -25% Zeitkosten

#### Wöchentliche Mega-Events (5-7 Tage)
- **Häufigkeit**: 1-2 pro Monat
- **Fokus**: Community-weite Herausforderungen
- **Beispiele**:
  - **Zeitkriege**: Fraktions-basierte Kämpfe zwischen Klassen
  - **Chronos-Erwachen**: Weltenboss mit Community-Lebenspunkten
  - **Zeitriss-Invasion**: Neue temporäre Dungeon-Mechaniken

### Event-Belohnungen

#### Standard-Belohnungen
| Event-Typ | XP-Bonus | Material-Bonus | Exklusive Belohnungen |
|-----------|----------|----------------|----------------------|
| Blitz | +100-200% | +50-150% | Event-Titel, kosmetische Items |
| Tages | +75-150% | +100-200% | Seltene Materialien, Event-Karten |
| Mega | +200-300% | +200-400% | Legendäre Titel, exklusive Skins |

#### Event-exklusive Inhalte
- **Temporäre Kartenvarianten**: Nur während Events verfügbar
- **Event-Titel**: Permanent, zeigen Teilnahme an besonderen Events
- **Kosmetische Upgrades**: Kartenrahmen, Partikeleffekte, UI-Themes
- **Lore-Erweiterungen**: Story-Fragmente und Charakterhintergründe

### Event-Mechaniken

#### Community-Herausforderungen
- **Globale Ziele**: Alle Spieler tragen gemeinsam bei
- **Belohnungsstufen**: Je mehr die Community erreicht, desto besser die Belohnungen
- **Beispiel**: "Besiegt 1 Million Gegner weltweit" → Freischaltung neuer Inhalte

#### Adaptive Events
- **Spieler-Level-Skalierung**: Events passen sich an Spielerstärke an
- **Klassenspezifische Varianten**: Verschiedene Ziele je nach gewählter Klasse
- **Progression-basierte Belohnungen**: Bessere Belohnungen für fortgeschrittene Spieler

```python
def apply_balance_adjustments(player_id, reward_type, base_amount):
    """Wendet Live-Balancing-Anpassungen an"""
    
    config_key = f"{reward_type}_multiplier"
    if config_key in BALANCE_CONFIG:
        multiplier = BALANCE_CONFIG[config_key]
        adjusted_amount = int(base_amount * multiplier)
        
        # Log für Telemetrie
        log_balance_adjustment(player_id, reward_type, base_amount, adjusted_amount, multiplier)
        
        return adjusted_amount
    
    return base_amount
```
