- **Pity-Timer ENTFERNT**: Pure RNG-Drops wie in Mo.Co

### 3. ✅ Event-System (Vollständig)

- **Mastery-Events**: Exklusive Events ab M5+

### 4. ✅ Mastery-System (Post-Level 25)
- **Endlose Progression**: M1, M2, M3... ohne Obergrenze
- **Klassenspezifische Boni**: Detailliert für alle 3 Klassen

- **Level-Gates**: Korrekt skaliert für neue Progression
