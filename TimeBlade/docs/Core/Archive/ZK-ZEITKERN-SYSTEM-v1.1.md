# Zeitklingen: Zeitkern-System (Ultra-vereinfacht)

## Zeitkern-System: Das Herzstück

**Grundprinzip:** 1 Zeitkern = 1 Level für eine Karte

### Zeitkern
- **Funktion**: Zufällige Karte +1 Level
- **Erhalt**: Kämpfe (häufig), Quests, Events
- **Maximum**: Karten bis Level 50

### Zeitkernkit  
- **Funktion**: Gezielte Kartenauswahl → +1 Level
- **Erhalt**: 3 Tagesquests = 1 Kit
- **Auswahl**: 2 Basiskarten + alle Varianten

### Automatische Gates
- **Level 10/20/30/40**: Automatische Seltenheits-Upgrades
- **Keine Materialkosten**: Gates passieren automatisch
- **Attribut-Boosts**: +10%/+20%/+30%/+45% zufällig

### Klassenstufen-Limit
- **Kartenlevel ≤ Klassenstufe × 2**
- Verhindert Über-Leveling

### Einschränkungen
- Karten an Gates können nicht gelevelt werden
- Karten am Klassenlimit können nicht gelevelt werden
- Karten auf Level 50 können nicht gelevelt werden

Das war's - maximale Vereinfachung erreicht.
